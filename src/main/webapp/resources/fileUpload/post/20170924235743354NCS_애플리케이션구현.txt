[애플리케이션 구현]

Java EE
- 개별적인 API를 묶어 놓은 표준 스펙이다.

- 자바를 이용한 서버사이드 API를 이용한 개발을 위한 플랫폼이다

- 포함된 API에는 Java Servlet, Java Server Pages, JDBC 등이 있다.


자바 가상 머신(JVM)의 특성
- 자바 바이트코드를 실행하는 주체
- 운영체제와 자바프로그램을 연결시켜주는 역할

- 플랫폼에 독립적으로 동작
- 인터프리터나 JIT 컴파일 방식으로 바이트코드를 실행할 수 있도록 함

- JRE에 포함되어 배포됨
- 가비지 컬렉션을 수행

- 과거 순수 인터프리터 방식으로 실행시간이 늦은 편이었으나 JIT(Just-In-Time) 컴파일을 구현하여 성능을 개선
